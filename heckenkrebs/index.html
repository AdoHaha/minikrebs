Konfiguration:

Das knackan der Accesspoints ist modular aufgebaut. Die Module befinden sich in /usr/lib/autowifi/plugins.

Eigene Module schreiben:

Eigene Module koennen einfach hinzugefuegt werden. Das autowifi Script ruft die Module in alphabetischer Reihenfolge auf.

Uebergabeparameter von autowifi sind: $1=ESSID $2=BSSID $3=CHANNEL $4=ENCRYPTION(off oder on) $5=WPA(0 oder 1) $6=WPA2(0 oder 1)
